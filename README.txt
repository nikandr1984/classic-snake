План разработки "Змейки" на Unity 

1. Подготовка ресурсов
+ Создать спрайты:  
  + Голова змеи  
  + Сегмент тела (префаб)  
  + Еда (префаб)  
  + Стены (опционально)  
+ Настроить камеру (Orthographic)  

2. Движение змейки 
+ Реализовать управление (WASD/стрелки)  
+ Сделать плавное перемещение по сетке  
+ Добавить рост при поедании еды (список сегментов)  

3. Механика еды  
+ Создать спавнер еды (случайные координаты)  
+ Проверка столкновения с едой (уничтожение + новый спавн)  

4. Коллизии и Game Over 
+ Столкновение с собой → проигрыш  
+ Столкновение со стенами → проигрыш 
+ Стена мигает красным при столкновении

5. UI и улучшения 
- Счетчик очков  
- Экран Game Over  
- Пауза / рестарт  

6. Дополнительно (по желанию)**  
- Звуки  
- Анимации  
- Разные виды еды  

7. Тестирование и доработки
